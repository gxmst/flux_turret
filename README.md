# Flux Turret - 高能光棱防御系统

![Minecraft 1.20.1](https://img.shields.io/badge/Minecraft-1.20.1-brightgreen.svg)
![Forge](https://img.shields.io/badge/Forge-47.4.10-orange.svg)

> "能量不仅是毁灭的工具，更是秩序的延伸。"

**Flux Turret (能量炮塔)** 是一款为您在 Minecraft 1.20.1 世界中提供尖端防御的模组。它不依赖传统的实弹弹药，而是通过高度集成的 **Forge Energy (FE)** 驱动，将纯粹的能量转化为划破黑暗的激光束。

## 🌌 技术巅峰：红警机制的完美复刻

本模组的核心在于实现了一套极其精密且稳定的**光棱连锁 (Prism Chaining)** 机制，其背后蕴含了深厚的技术底蕴：

### 🛠️ 图论算法：梯度流中继 (Gradient Flow Relay)
为了实现像《红色警戒2》中那样自然且高效的能量传导，我们舍弃了笨重的 BFS 搜索，转而采用一种受物理学启发的**梯度流算法**。
- **状态传播：** 每个炮塔根据与目标的距离自动计算“电势深度”。
- **动态寻路：** 能量流会自动寻找深度最小的路径，实现从 C -> B -> A -> 目标的完美中继。
- **无限延伸：** 只要链路不断，信号可以在毫秒级时间内穿透整个阵列。

### 🔒 分布式锁：循环与超载控制
在复杂的炮塔网络中，防止能量“鬼打墙”是最大的挑战。
- **防止死循环：** 通过单向深度校验实现的“逻辑锁”，确保能量永远只流向离怪最近的 Master 塔。
- **负载均衡：** 引入了节点接入上限限制，当主塔饱和时，多余的能量会自动流向旁路节点，形成壮观的树状拓扑网络。

### 📡 异步同步：跨越时空的视觉一致性
解决服务端逻辑与客户端渲染之间的毫秒级时差是我们的核心课题。
- **数据包强同步：** 每一发脉冲的开火时间 (`lastFireTime`) 和目标坐标都会经过精密封装，强制推送到周围玩家。
- **视觉残留缓冲：** 引入了客户端渲染预测与残留逻辑，即便怪物在瞬间被蒸发，激光依然会射向其“最后遗址”，提供极致的打击反馈。

### 🎨 渲染管线：动态 3D 光束建模
放弃了简陋的线性绘制，我们重构了渲染管线：
- **体积感建模：** 使用相互垂直的双面 Quad 面片模拟 3D 体积光。
- **动态波形：** 基于正弦波的脉冲抖动效果，赋予激光不稳定的电浆质感。
- **能量喷涌：** 由细变粗的渐变设计，完美展现了能量从汇聚到爆发的视觉冲击。

## 🛠️ 合成与使用

1.  **建造底座：** 使用铁块构筑稳固的防御平台。
2.  **植入核心：** 在顶端嵌入钻石作为能量聚焦核心。
3.  **注能：** 接入任何支持 Forge Energy 的电源或线缆。
4.  **阵列：** 建议以 10-12 格为间距成排布置，开启连锁集火模式。
